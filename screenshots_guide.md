# VigilTip - 应用截图指导

## 📸 截图要求

### 技术规格
- **格式**: PNG 或 JPEG
- **数量**: 至少2张，最多8张
- **尺寸**: 手机截图 (16:9 或 18:9 比例)
- **语言**: 建议使用英文截图（主要市场）

## 🎯 推荐截图顺序

### 1. 启动页面 (必需)
```
📱 内容: VigilTip启动界面
🎨 重点: 应用图标 + "VigilTip" + "Think before you act"
⏰ 时间: 3秒启动动画完成时
📐 构图: 居中显示，包含完整界面
```

### 2. 相机页面 (必需)
```
📱 内容: 收据扫描界面
🎨 重点: 相机预览 + 拍照按钮 + 手动输入选项
⏰ 时间: 相机初始化完成后
📐 构图: 显示完整的相机界面和按钮
```

### 3. 结果页面 (必需)
```
📱 内容: 小费建议结果
🎨 重点: 三个小费选项 + 金额显示 + 百分比
⏰ 时间: 显示小费建议时
📐 构图: 清晰显示小费金额和百分比
```

### 4. 设置页面 (可选)
```
📱 内容: 设置界面
🎨 重点: 隐私政策入口 + 应用信息
⏰ 时间: 设置页面打开时
📐 构图: 显示设置选项列表
```

### 5. 手动输入页面 (可选)
```
📱 内容: 手动金额输入界面
🎨 重点: 输入框 + 计算按钮
⏰ 时间: 手动输入界面打开时
📐 构图: 显示输入界面和键盘
```

## 🎨 截图技巧

### 准备工作
```
📱 设备: 使用真实Android设备
🌐 网络: 确保网络连接稳定
🔋 电量: 充足电量避免低电量提示
📵 通知: 清理通知栏避免干扰
```

### 拍摄时机
```
⏰ 启动页面: 应用启动后2-3秒
📸 相机页面: 相机完全初始化后
💡 结果页面: 显示小费建议时
⚙️ 设置页面: 页面完全加载后
```

### 构图要点
```
🎯 居中构图: 重要内容居中显示
📐 完整界面: 确保界面元素完整
🌈 色彩鲜明: 确保色彩准确
🔍 清晰度: 高清截图，无模糊
```

## 📱 具体操作步骤

### Android设备截图
```bash
# 方法1: 使用ADB命令
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png

# 方法2: 设备快捷键
电源键 + 音量下键 (同时按下)
```

### 截图后处理
```
🖼️ 裁剪: 裁剪到应用界面边界
📏 尺寸: 确保符合Google Play要求
🎨 优化: 适当调整亮度和对比度
📝 命名: 按顺序命名 (screenshot_1.png, screenshot_2.png...)
```

## 🔍 截图检查清单

### 每张截图检查
```
✅ 界面完整: 所有UI元素都可见
✅ 文字清晰: 文字内容可读
✅ 色彩准确: 颜色显示正常
✅ 无干扰: 没有通知栏或其他干扰
✅ 尺寸合适: 符合Google Play要求
✅ 内容相关: 展示核心功能
```

### 整体检查
```
✅ 数量充足: 至少2张，建议4-6张
✅ 顺序合理: 按使用流程排序
✅ 语言一致: 使用同一种语言
✅ 风格统一: 保持视觉风格一致
✅ 功能覆盖: 覆盖主要功能点
```
